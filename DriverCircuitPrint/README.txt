PCB Print Driver

First version of LED Display was wired manually. This was realy an intensiv work and no motivation to do it twice.
A PCB development for the driver circuit was the solution. The idea is to separate the driver PCB for each digit. The usage in other projects should be possible. The prototype (prototype-develop-attempt.JPG) wireing from the first attempt was a good base to start with KiCad. The schematic was drawn quickly (Driver-schema.pdf). A PCB was printed and the wireing with an prototype test digit succeeded (test-digit.JPG).
